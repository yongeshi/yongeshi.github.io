RPG Game
A rpg-style escape game with sprite animation written in Java and JavaFX for the GUI.

How to Play the Game:
The object of the game is to escape the room. The player must obtain the key in order to exit the room. Use the arrow keys to move the player around. Move your character around the room and press the space bar on objects to check if the key is there. Once the key is obtained, the player may exit the room via the stairs.
Android Application
MapCovid
MapCovid is an android mobile application made with Android Studio in Java. MapCovid displays COVID related news and infection rates within the Los Angeles county. It also shows nearby COVID test locations and the user’s past travel history path if locations are turned on.

App Features:
-loading screen, onboarding instructions, geofencing, push notifications when traveling, path tracking and ability to delete history, an interactive map (color-coded by number of active cases of covid in the specific region), covid testing locations, up-to-date information on COVID in the form of relevant Twitter feeds, the latest news articles, and case statistics

In Ur Area
A fun frontend/backend-focused website to search for information on nearby restaurants with the ability to create an account and login to save your favorite restaurants.

Web Features:
